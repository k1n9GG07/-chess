# Шахматный проект (Chess Project)

Данный проект представляет собой веб-приложение для игры в шахматы, разработанное с использованием современных подходов к программированию (ООП, SOLID) и методологии верстки БЭМ.

## Примененные принципы SOLID

1. **Single Responsibility Principle (Принцип единной ответственности)**:
   - Каждый класс в проекте выполняет строго одну задачу. 
   - `Piece.js` и его наследники отвечают только за логику движения конкретных фигур.
   - `Board.js` отвечает за управление состоянием игрового поля и его инициализацию.
   - `Game.js` управляет игровым процессом, ходами и взаимодействием с пользователем (UI).
   - `main.js` отвечает за запуск приложения.

2. **Open/Closed Principle (Принцип открытости/закрытости)**:
   - Базовый класс `Piece` открыт для расширения (можно легко добавить новые типы фигур, например, "Ферзь-дракон"), но закрыт для модификации. Основная логика доски (`Board`) не меняется при добавлении новых фигур.

3. **Liskov Substitution Principle (Принцип подстановки Барбары Лисков)**:
   - Все наследники класса `Piece` (`Pawn`, `Rook`, `Knight` и др.) могут быть использованы везде, где ожидается объект типа `Piece`, без нарушения логики программы.

4. **Dependency Inversion Principle (Принцип инверсии зависимостей)**:
   - Класс `Game` зависит от абстракции `Piece`, а не от конкретных реализаций, используя общие методы `isValidMove` и `getPossibleMoves`.

## Принципы ООП

1. **Инкапсуляция**: Все свойства фигур (цвет, тип, символ) и методы логики скрыты внутри соответствующих классов.
2. **Наследование**: Все шахматные фигуры наследуются от общего базового класса `Piece`, переиспользуя его код.
3. **Полиморфизм**: Метод `isValidMove` реализован по-разному в каждом классе фигуры, что позволяет вызывать его универсально для любой фигуры на доске.
4. **Абстракция**: Использование базового класса `Piece` как шаблона для всех фигур позволяет работать с "абстрактной фигурой", не вдаваясь в детали её типа.

## Методология БЭМ (Блок-Элемент-Модификатор)

Верстка выполнена строго по методологии БЭМ:
- **Блоки**: `chess-board`, `game`, `header`, `footer`.
- **Элементы**: `chess-board__cell`, `game__status`, `header__title`.
- **Модификаторы**: `chess-board__cell--white`, `chess-board__cell--black`, `chess-board__cell--selected`, `chess-board__cell--possible-move`.

## Функциональность программы
- Корректная реализация ходов всех шахматных фигур.
- Проверка на "Шах", "Мат" и "Пат".
- Подсветка выбранной фигуры и доступных ходов.
- История ходов с отображением взятых фигур.
- Возможность начать игру заново.

## Инструкция по запуску
1. Склонируйте репозиторий.
2. Откройте `index.html` в браузере.
3. Для работы ES6 модулей рекомендуется использовать локальный сервер (например, расширение Live Server в VS Code).
